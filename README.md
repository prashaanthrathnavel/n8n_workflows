## 📂 Workflow Projects

### 🖨️ 1. AI Print App — Smart Xerox Automation
> Intelligent print-on-demand system for students and working professionals

**What it does:**
- User submits print request via webhook
- Groq AI (LLaMA 3.3 70B) analyzes document and suggests best print settings
- Finds nearest Xerox shop automatically
- Sends WhatsApp confirmation with price estimate and AI tips

**Tech Stack:** n8n · Groq API · LLaMA 3.3 70B · Twilio WhatsApp · OpenStreetMap · JavaScript

## 🚀 How to Use These Workflows

1. Install [n8n](https://n8n.io) locally or on cloud
2. Open n8n → Click `+` → `Import from file`
3. Select any `.json` workflow file from this repo
4. Add your own API credentials
5. Activate and test!
